#  EcommerceBookStore

A full-stack e-commerce web application for buying and selling books with user and admin dashboards.

##  Features

### User Features
-  Browse book catalog with search functionality
-  View detailed book information
-  Add books to shopping cart
-  Place orders
-  User account management
-  Order history tracking

### Admin Features
-  Dashboard with analytics
-  Manage book inventory (Create, Read, Update, Delete)
-  User management
-  Order management
-  Sales tracking

##  Tech Stack

### Backend
- **Java 11+**
- **Spring Boot 2.x/3.x** - Web framework
- **Spring Data JPA** - Database ORM
- **Maven** - Dependency management

### Frontend
- **HTML5** - Markup
- **CSS3** - Styling
- **Bootstrap 5** - Responsive UI framework
- **Thymeleaf** - Server-side templating

### Database
- **MySQL/PostgreSQL** - Relational database

### Deployment
- **Docker** - Containerization
- **Render/Koyeb** - Cloud hosting
